CSS Szelektorok

A CSS-szelektorok segítségével meghatározott HTML-elemeket célozhatunk meg, hogy stílusokat alkalmazzunk rájuk.
A CSS középszintű felhasználói megértik, hogyan használhatnak összetettebb szelektorokat, például attribútum-szelektorokat,
pszeudo-osztályokat és pszeudo-elemeket, hogy pontosabb és célzottabb stílusokat hozzanak létre.

Box Model

A dobozmodell az a mód, ahogyan a CSS a HTML-elemeket téglalap alakú dobozokként kezeli, tartalommal, kitöltéssel, keretekkel és margókkal.
A CSS középszintű felhasználói megértik, hogyan használhatják a dobozméret tulajdonságot, valamint a kitöltés, a keret és a margó különböző értékeit, hogy rugalmasabb és érzékenyebb elrendezéseket hozzanak létre.

Css layouts

A CSS elrendezés a HTML-elemek elrendezése az oldalon. A CSS középszintű felhasználói megértik, hogyan használhatják a display tulajdonságot,
a pozicionálást és a flexboxot, hogy összetettebb és dinamikusabb elrendezéseket hozzanak létre.
