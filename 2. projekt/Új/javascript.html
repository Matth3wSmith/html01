JavaScript

,,A JavaScript programozási nyelv egy objektumorientált, prototípus-alapú szkriptnyelv, amelyet weboldalakon elterjedten használnak.”

A Javascript programok windows alapú környezetben futtathatók továbbá a wscript.exe és a cscript.exe segítségével is.

A Microsoft 1995-ben kifejlesztette az Internet Explorert ami miatt a Netscape-pel böngészőháborúba keveredtek.



Később a Microsoft kifejlesztette a Netscape Navigator Javascript-feldolgozójának forásskódjával a sajátját, a JScriptet.

Bár a nyelvet szabványosították, mégis részben különbözően implementálják a JavaScriptet a különböző böngészők.



A 2000-es évek elején az Internet Explorer dominanciája idején a kliensoldali szkriptelés stagnált. Ez 2004-ben kezdett megváltozni, amikor a Netscape utódja, a Mozilla kiadta a Firefox böngészőt. A Firefoxot sokan jól fogadták, jelentős piaci részesedést szerzett az Internet Explorertől.

A Google 2008-ban mutatta be Chrome böngészőjét, a V8 JavaScript motorral, amely gyorsabb volt, mint versenytársai. A kulcsfontosságú újítás a just-in-time compilation (JIT) volt, így más böngészőgyártóknak is át kellett javítaniuk motorjaikat a JIT számára.
